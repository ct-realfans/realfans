為什麼是現在 · Why Now

WHY.1

AI 讓個人化變便宜

2024-25 以前，一封個人化訊息要雇一個小編半天。現在 Claude Sonnet 一次 NT$0.5、2 秒產出。個人化成本 = 10,000× 便宜。

WHY.2

FTC 新規 + 公平會升級

美國 FTC 2024-10 生效：每則假評論罰 USD $51,744。台灣公平會 2025 起鎖定電商平台與代操公司。粉絲多的存在進入倒數計時。

WHY.3

LINE 滲透率 95% · 但海外 SaaS 沒整合

Podium / Birdeye / Yext 都不做 LINE。台灣小商家最主要的顧客通路 = 我們天然的護城河。

WHY.4

Google Business Review Request API 開放

2025 起可合法用官方 API 邀評（whitelist）。我們直接站在 Google 認可的一側。

護城河 · MOAT

海外 SaaS 不會做 · 本土代操也追不上

MOAT.1

繁體中文 × 台灣在地 prompt 庫

春節、中秋、颱風假、地震關懷 — 海外 SaaS 寫不出台灣人會讀的語氣。我們每一則互動都在訓練品牌語氣模型。

MOAT.2

LINE OA Native + POS 深度整合

iCHEF、Square、PosCat、iPOS 等台灣 POS 直連，一旦切進去，替換成本極高（顧客資料、歷史評論、品牌語氣都搬不走）。

MOAT.3

合規背書 · compliance_events 稽核軌跡

每次 AI 操作都寫 audit log，連 prompt 版本都留底。公平會或 Google 查帳能秒匯出。粉絲多根本拿不出這種東西。

MOAT.4

資料飛輪 · 越多店越準

每家店的顧客偏好 × 回饋成效 → 訓練越來越精準的在地化 AI。1,000 家店後，新店第一天的邀評轉換率就贏過單打獨鬥 6 個月的店。

MOAT.5

相對粉絲多：正當性

粉絲多永遠不能上 PChome 廣告、不能上電視、不能接政府標案。我們可以。這是永久性單向優勢。
