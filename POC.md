## 招募條件（3 家垂直）

### POC #1 · 咖啡廳 / 早午餐（量化路徑 1：Google 星等）

**理想店家**：
- 台北/新北獨立店、月營收 NT$300-600K
- 現有 Google 評論 30-100 則，星等 4.0-4.4
- 老闆願意分享月營收趨勢（同意保密協議後）
- 有 LINE OA 但**未在積極經營**

**為什麼選這種**：
- Luca 2016 研究就在餐飲垂直，可直接對照
- Google 星等提升最容易量化看到
- 3 個月足以看到變化

### POC #2 · 社區診所（量化路徑 2：LINE 回訪）

**理想店家**：
- 牙科 / 皮膚科 / 兒科 / 中醫
- 月初診 30-80 人
- 已有病患 LINE 聯絡
- 有推薦預算或想省廣告費

**為什麼選這種**：
- 客單價高（NT$2K-10K），ROI 倍數最漂亮
- LINE 回診提醒效果易驗證
- 評論對「選診所」決策權重極大

### POC #3 · 美容 / 美髮工作室（量化路徑 3：綜合）

**理想店家**：
- 單人到 3-5 人的小工作室
- 月營收 NT$150-300K
- 熟客回頭率是生命線
- 已用 LINE 預約

**為什麼選這種**：
- 單點老闆決策快，容易推
- 小數據量好看因果關係
- Starter 方案驗證（低價位客群）

## 招募 pitch（給商家的提案）

### 我們給你
1. ✅ **免費用 Pro 方案 3 個月**（價值 NT$11,970）
2. ✅ **創辦人親自 onboard**：匯客戶資料、設 LINE、教 UI
3. ✅ **週一次 15 分鐘 check-in**
4. ✅ **結束後選擇**：喜歡就付月費續用、不喜歡零負擔

### 你給我們
1. 📊 **baseline 數據**（近 90 天）：
   - Google 評論截圖
   - 月營收總額（可遮掉敏感欄位、只要區間）
   - LINE OA 好友數
2. 📊 **使用期每週填一次表**：該週新增評論、LINE 推播效果、客訴
3. 📝 **3 個月結束**：公開或匿名案例研究（名字可遮）
4. 🎥 **1 分鐘感想影片**（可選，會加 6 個月免費）

### 為什麼老闆會答應
- 3 個月免費 = 省 NT$11,970
- 看到粉絲多的法律風險，想有正當選擇
- 老闆面對「我家的真顧客在哪」這個問題想很久了
- 我們是創辦人親自服務

## 測量方法論

### Baseline（簽約當週）

| 指標 | 怎麼收 | 狀態 |
|---|---|---|
| Google 星等 / 總評論數 | 截圖 Google Business Profile | 必收 |
| 過去 90 天月新增評論 | Google Business 後台 | 必收 |
| LINE OA 好友數 | LINE Manager | 必收 |
| 月營收（近 90 天均值） | 老闆自述 / POS 截圖 | 必收 |
| 社群帳號粉絲數（IG/FB） | 截圖 | 加分 |
| 是否使用粉絲多等灰產 | 問卷 | 加分 |

### 週度追蹤（12 週）

每週一填：
- 該週新增 Google 評論數 + 新評論星等分佈
- 該週發送邀評數（系統自動記錄）
- 該週封鎖率（LINE 退訂數）
- 該週老闆覺得最有感的「我感覺有差的事」

### 90 天後結果（POC 結束）

對比 baseline 測：
- Δ Google 星等
- Δ 月評論數
- Δ LINE 好友數
- Δ 月營收（老闆願意分享的話）
- 邀評轉換率（系統數據）
- 封鎖率
- 老闆 NPS（0-10 分）

## POC 協議要點（給商家看的）

1. **數據用途**：匿名或具名（商家選）呈現在 RealFans 行銷與募資材料
2. **隱私**：營收數字以區間呈現、客戶姓名不外流
3. **中止**：雙方任何一方可無條件 30 天內中止
4. **後續使用**：3 個月後若商家續用付月費，早鳥價 NT$2,990/月（原 Pro 75 折）鎖 12 個月
5. **競業**：商家不得 3 個月內轉用粉絲多等灰產服務（不然數據失真）

## 招募管道

1. **創辦人人脈**：直接問認識的商家朋友（最快）
2. **LINE 社團**：台北早午餐社、美髮師社群
3. **地推**：帶平板直接進店示範（3 分鐘展示）
4. **冷 email**：爬 Google Maps 4.0-4.4 星的商家（1-2% 回覆率）
5. **KOL 合作**：跟商業 podcast / 小店主 YouTuber 合作引介

**第一批目標**：2-3 週內完成 3 家簽約。
